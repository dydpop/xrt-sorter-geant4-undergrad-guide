# Notice

This repository is a public learning and demonstration package for the undergraduate Geant4 XRT mineral sorting simulation project.

The project is shared so team members can:

- understand the project background
- learn the code and documentation structure
- reproduce the undergraduate-level results
- prepare discussion, review, and presentation materials

The project is not shared as a permissive open-source package. Public visibility does not mean permission to copy, rebrand, republish, sell, or submit this work as another person's independent project.

The reported `0.98` accuracy is limited to the simulated coarse absorption-group classification task included in this repository. It must not be described as field-device performance or universal mineral sorting performance.
