# 本科级最终复核说明

## 复核结论

当前公开仓库达到了本科级“仿真系统 + 数据链路 + 基础分类验证 + 文档封装”的交付目标。仓库包含 Geant4 C++ 仿真代码、六材料公开配置、Python 分类脚本、紧凑验证证据包、图表、论文式说明和组员学习指南。

## 已完成证据链

项目完成了以下链路：

```mermaid
flowchart LR
    A["六材料配置"] --> B["Geant4 仿真"]
    B --> C["events.csv"]
    C --> D["虚拟样本"]
    D --> E["训练/测试拆分"]
    E --> F["基础分类"]
    F --> G["混淆矩阵与论文"]
```

`results/undergrad_validation/` 是当前最重要的复核入口。它显示每种材料 5000 个 events、50 个虚拟样本；训练集 150 个样本、测试集 150 个样本。当前证据包中，阈值法、单特征 Logistic Regression 和三特征 Logistic Regression 的测试 accuracy 分别为 `0.9800`、`0.9867`、`0.9933`。

## 可以成立的表述

可以说：本项目完成了基于 Geant4 的 XRT 矿物分选仿真原型，能够输出事件级数据，并用 Python 将仿真数据转化为可解释特征和本科级分类验证结果。

可以说：在当前六材料、固定几何、仿真数据和粗粒度吸收组任务中，测试集最高 accuracy 为 `0.9933`。

可以说：当前分类结果来自明确的训练/测试拆分，不是训练集准确率。

## 不应成立的表述

不应说：该项目已经验证真实设备效果。

不应说：该模型能分类世界上所有矿物。

不应说：当前结果可以直接用于工业在线控制。

不应说：`0.9933` 是普适常数。Geant4 仿真有随机性，重新生成事件 CSV 后结果可能小幅波动。

## 后续改进方向

如果要继续提升研究等级，建议优先补充固定随机种子或多随机种子重复实验、不同厚度和粒径配置、混合材料样本、按独立 run 拆分的测试集、交叉验证，以及真实设备数据对照。
