# 中文摘要

本项目设计并实现了一个基于 Geant4 的 X 射线透射（XRT）矿物分选仿真原型。系统使用 C++ 构建 X 射线源、矿物样本、探测器、事件记录和命中记录，使用配置文件控制材料、几何和源项参数，并通过 Python 对仿真事件数据进行样本构造、特征提取和基础分类验证。公开复现实验包含 Quartz、Calcite、Orthoclase、Albite、Dolomite、Pyrite、Hematite、Magnetite、Chalcopyrite 和 Galena 十种单一材料，每种材料运行 5000 个仿真事件，每 100 个事件聚合为 1 个虚拟样本，共得到 500 个虚拟样本。按每种材料前 25 个样本训练、后 25 个样本测试的规则，形成 250 个训练样本和 250 个测试样本。当前证据包中，基于主 gamma 透射率的阈值法在测试集上达到 0.9840 accuracy，单特征 Logistic Regression 达到 0.9920，三特征 Logistic Regression 达到 0.9960。结果表明，在当前十材料、固定几何、仿真数据和粗粒度吸收组任务中，XRT 事件数据能够形成有区分度的物理相关特征。补充材料级诊断显示，直接预测十个材料标签时主方法 top-1 accuracy 仅为 0.464，所有材料级验收条件均未通过，因此本文结论不代表真实设备性能、复杂矿流表现、十材料物种识别成功或对所有矿物的普适分类能力。
