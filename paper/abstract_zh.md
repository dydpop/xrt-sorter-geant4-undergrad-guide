# 中文摘要

本项目围绕 X 射线透射矿物分选过程，设计并实现了一个基于 Geant4 的仿真原型系统。系统包括 X 射线源、矿石样本、探测器响应、事件数据输出和 Python 数据分析流程。通过材料配置、能谱输入和探测器统计，项目生成了可用于基础分类验证的仿真数据。

在数据分析部分，项目提取了透射率、能量沉积、直接命中和散射命中等特征，并使用阈值法和 Logistic Regression 等基础方法完成粗粒度吸收组分类。在当前仿真数据和当前分类任务中，最佳 accuracy 达到 `0.98`。该结果说明仿真系统可以产生有区分度的物理相关特征，能够支撑本科项目层面的系统实现和初步算法验证。

本文结论仅适用于本仓库公开的仿真设置和粗粒度分类任务，不代表现场设备效果或所有矿物条件下的通用表现。
