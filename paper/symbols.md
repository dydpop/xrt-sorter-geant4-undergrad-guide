# 符号与缩写

| 符号或缩写 | 含义 |
| --- | --- |
| XRT | X-ray Transmission，X 射线透射 |
| Geant4 | 粒子与材料相互作用仿真工具包 |
| CSV | Comma-Separated Values，表格数据文件 |
| Edep | Energy deposition，能量沉积 |
| accuracy | 分类正确率 |
| low_absorption | 低吸收组 |
| high_absorption | 高吸收组 |
| Logistic Regression | 逻辑回归，一种基础分类方法 |
| CMake | C++ 项目构建工具 |
